# 🖐️ Давай Меняться

Приложение для **обмена и дарения детских игрушек**. Продажа запрещена — только обмен, дар или аренда. Помогает экономить бюджет, избегать бардака игрушек и учит детей бережному потреблению.

🔗 **Живая версия:** https://kusov777.github.io/davai-menyatsya/
📲 На телефоне: открыть ссылку → меню браузера → «Добавить на главный экран» → появится иконка-ладонь.

---

## Как открыть прототип

**Быстро (посмотреть):** дважды кликните `index.html` — откроется в браузере. Работает всё, кроме «установки на телефон» (для неё нужен запуск через сервер, см. ниже).

**Как настоящее приложение (PWA, с установкой на телефон):**
```
cd "Давай меняться 1"
python -m http.server 8753
```
Откройте на телефоне `http://<IP-компьютера>:8753` в одной Wi-Fi-сети → меню браузера → «Добавить на главный экран». Появится иконка-ладонь, запускается без адресной строки.

## 🛒 Дорожная карта в App Store и Google Play

Прототип = готовый UI/UX. Чтобы оказаться в магазинах, нужно ещё 3 вещи. Честно, без приукрашивания:

### 1. Что нужно ОБЯЗАТЕЛЬНО (для любого пути)
| Что | Зачем | Деньги / условия |
|---|---|---|
| **Бэкенд** | реальные пользователи, БД лотов, загрузка фото, чат, отзывы, модерация (запрет продаж/LEGO) | от $0 (Supabase/Firebase free) до платных по росту |
| **Google Play Console** | публикация в Google Play | **$25 единоразово**. Сборка возможна **на Windows** |
| **Apple Developer** | публикация в App Store | **$99/год** + **нужен Mac** (или облачный Mac: Codemagic/Appflow) |
| **Политика конфиденциальности + согласия** | данные детей → строгий регламент | GDPR-K (ЕС), 152-ФЗ + реестр операторов ПДн РКН (если есть пользователи РФ), Apple Kids / Google Families |

⚠️ **Детские данные + локация** — самая чувствительная зона. Apple и Google проверяют такие приложения строже (родительский контроль, запрет сторонней аналитики, согласия). Это нужно заложить в архитектуру с самого начала.

### 2. Как упаковать (3 маршрута)

**A. Capacitor — рекомендую.** Оборачивает этот же веб-код в нативное iOS+Android приложение, один код на оба стора. Android собираете на Windows → Google Play; iOS — нужен Mac или облачный Mac. Переиспользует 100% уже сделанного.

**B. PWABuilder (Microsoft, бесплатно).** Берёт размещённый PWA-адрес и генерирует готовый пакет для Google Play и проект для App Store. Самый быстрый путь для Android.

**C. Полная нативка (React Native/Expo или Flutter).** Максимально «родное» приложение, лучше проходит ревью Apple, но это переписывание с нуля (недели) + бэкенд. Оправдано после подтверждения спроса.

### 3. Рекомендуемая последовательность (lean, минимум денег вперёд)
1. **Сейчас, $0:** установить PWA на телефон, показать 10–20 родителям → проверить, что обменом реально пользуются.
2. **Есть спрос →** поднять бэкенд (Supabase) + оформить юр./политику для детских данных.
3. **→** упаковать через Capacitor: Android в Google Play ($25), затем iOS (Mac/облако + $99).

## Файлы проекта
- `index.html` — всё приложение (один файл: вёрстка, стили, логика)
- `manifest.json`, `sw.js` — PWA (установка на телефон, офлайн-кэш)
- `icons/` — иконка-ладонь (192/512/1024/apple-touch)
- `gen_icons.py` — генератор иконок (если нужно перерисовать)

## Технологии
HTML/CSS/JS без сборки · Leaflet + OpenStreetMap (карта, бесплатно, без ключа) · шрифты Unbounded + Nunito · данные в localStorage.
